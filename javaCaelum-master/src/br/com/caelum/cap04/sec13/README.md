

** 2 - Por que o modo acima é extremamente mais lento para calcular a série do que o modo iterativo (que se usa um laço)? **

Os metodos recursivo necessitam registrar o estado atual do processamento de forma a continuar de onde parou durante cada execucao subordinada do processo recursivo, dessa forma há um consumo de mais memoria e tempo que um metodo iterativo.
