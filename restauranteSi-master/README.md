####Projeto RestauranteSI

Sistema em Java que representa algumas das atividades de um restaurante. Foi desenvolvido para exercitar alguns conceitos de Orientação a Objetos.

#####Alguns do metodos estaticos que a classe Arrays em _java.util.Arrays_ possui genericamente são:

- asList 

Devolve uma lista a partir de um array

- binarySearch

Efetua uma busca binaria

- copyOf

Expande um array fazendo copia dos valores no estado anterior do array

- sort

Ordena um array de um determinado tipo

#####Utilizando representação de dinheiro em Java:

Geralmente dados monetarios devem ser preferivelmente armazenados como sendo BigDecimal, long ou int; 

Uma alternativa interessante para representar numeros monetarios que não seja um tipo primitivo, mas que se utiliza do formato BigDecimal seria a MoneyApi, descrita no artigo fonte.
